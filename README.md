# Affective Computing Project

## Description
This project leverages affective computing principles to analyze human emotions and develop applications that can respond to users' emotional states. It aims to create a smarter interaction between users and computers based on emotional feedback.

## Features
- Emotion recognition from text and speech.
- Real-time interaction capabilities.
- Visualization of emotions through graphs and charts.
- API for easy integration with other applications.
- Modular design for easy enhancements and modifications.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/MinhalShafiq/affective_computing_project.git
   cd affective_computing_project
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Usage
- To start the application, run the main script as shown in the installation instructions.
- Follow the on-screen prompts to analyze text or audio for emotional content. 

## Project Structure
- `app/`: Contains the main application logic.
- `data/`: Stores datasets for training and testing.
- `models/`: Machine learning models used for emotion recognition.
- `tests/`: Unit tests for the application.
- `requirements.txt`: Lists required packages for the project.
